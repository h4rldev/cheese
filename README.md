# Cheese

A simple WIP Immediate Mode GUI library for C, made with [htils](https://github.com/h4rldev/htils), [harfbuzz](https://github.com/harfbuzz/harfbuzz), and [freetype](https://freetype.org/).

## Status

Quite broken, can render widgets, although barely, font rendering is in its infancy, and i can't properly dynamically resize the font properly.
